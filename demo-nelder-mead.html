Nelder-Mead Algorithm Visualization

Nelder-Mead is a derivative-free optimization method that finds a local minimum by iteratively modifying a simplex — a geometric shape with one more vertex than the space has dimensions. In 2D, the simplex is a triangle. Each step reflects, expands, contracts, or shrinks the triangle to move it toward the minimum.

pseudocode
Initialize simplex with three points Evaluate f at each vertex Sort: f(u) ≤ f(v) ≤ f(w) Compute centroid c = (u + v) / 2 Reflect: r = c + α(c − w), α = 1 Evaluate f(r) If f(u) ≤ f(r) < f(v): replace w ← r Else if f(r) < f(u): Expand: e = c + γ(r − c), γ = 2 Evaluate f(e) If f(e) < f(r): replace w ← e Else: replace w ← r Else (f(r) ≥ f(v)): Contract: p = c + ρ(w − c), ρ = 0.5 Evaluate f(p) If f(p) < f(w): replace w ← p Else: shrink all toward u, σ = 0.5 Check convergence Repeat ↑
visualization · f(x, y) = (x − 1)² + 0.7(y + 1)²
